FinNova AI 💰🤖

OVERVIEW

FinNova AI is an AI-Powered Personal Finance Management System developed as a Final Year Project. The application helps users manage daily financial activities, track income and expenses, plan budgets, monitor savings goals, and receive intelligent financial recommendations.

The project combines modern web technologies with AI-driven insights to provide users with a smart and interactive financial dashboard.

---

PROBLEM STATEMENT

Many individuals struggle to manage personal finances effectively due to:

• Lack of spending visibility

• Poor budgeting habits

• Difficulty tracking expenses

• Absence of financial goal monitoring

• No intelligent recommendations for saving money

FinNova AI addresses these challenges through a centralized finance management platform.

---

OBJECTIVES

• Track income and expenses efficiently

• Manage monthly budgets

• Set and monitor savings goals

• Visualize financial data through charts

• Generate AI-based spending recommendations

• Predict future expenses based on spending patterns

KEY FEATURES

1. User Authentication

• User Registration

• User Login

• Secure Access Management

2. Transaction Management

Users can:

• Add Transactions

• Edit Transactions

• Delete Transactions

Each transaction contains:

• Category

• Amount

• Merchant

• Payment Method

• Transaction Type

3. Budget Planner

• Set Monthly Budgets

• Monitor Spending

• Track Budget Utilization

4. Savings Goal Tracking

• Set Savings Goals

• Monitor Goal Progress

• View Remaining Amount Required

5. Financial Health Score

The system evaluates:

• Spending Behaviour

• Budget Adherence

• Savings Performance

and generates a Financial Health Score.

6. AI Spending Recommendations

FinNova AI analyzes:

• Spending Categories

• Budget Utilization

• Savings Goal Progress

• Predicted Expenses

and generates recommendations such as:

• Reduce spending in specific categories

• Save more towards financial goals

• Control unnecessary expenses

Example:

Reduce Food expenses by ₹2000 and save ₹3000 more to achieve your savings goal.

7. Expense Prediction

The AI module predicts future expenses using historical transaction data.

Benefits:

• Better Planning

• Budget Preparation

• Financial Awareness

8. Interactive Dashboard

The dashboard displays:

• Total Income

• Total Expense

• Current Balance

• Transaction Count

• Budget Status

• Savings Progress

9. Data Visualization

• Pie Chart – Category-wise Spending Distribution

• Bar Chart – Spending Overview

• Monthly Trend Chart – Spending Patterns Over Time

10. Search and Filter System

Users can:

• Search Transactions

• Filter Income Transactions

• Filter Expense Transactions

11. Report Generation

PDF Export:
Generate financial reports in PDF format.

Excel Export:
Generate financial reports in Excel format.

12. Dark and Light Mode

Users can switch between Dark Theme and Light Theme for a better experience.
